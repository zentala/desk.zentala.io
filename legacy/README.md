# Legacy Website

This directory contains the older static website that previously lived at the repository root.

## What is here

- `index.html` and `style.css` for the Bootstrap-based legacy site
- `images/` referenced by the old landing page
- `enclosure/` hardware enclosure source files linked from the page
- `notes/` historical supporting material

## Purpose

This content is preserved for reference, migration, and historical context.
It is no longer the active website implementation.

## Archived Markdown

- Original root README: `README.root-original.md`
- Original root CLAUDE guidance: `CLAUDE.root-original.md`
- Original Astro starter README: `README.astro-starter-original.md`

## Local usage

- Open `legacy/index.html` in a browser to inspect the old site.
- Keep relative paths unchanged inside `legacy/` so the page remains self-contained.
