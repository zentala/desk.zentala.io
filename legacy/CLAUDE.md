# CLAUDE.md - Legacy Site Guidance

## Scope

This file applies only to the preserved legacy website in `legacy/`.

## Purpose

Historical static website and supporting materials kept for reference, migration, and archival value.

## Contents

- legacy landing page: `index.html`
- legacy stylesheet: `style.css`
- legacy images: `images/`
- enclosure files: `enclosure/`
- historical notes and archived markdowns: `notes/` and `*.original.md`

## Handling Rules

- Preserve content unless there is an explicit migration or cleanup decision
- Prefer moving or copying old material over rewriting it in place
- Keep relative paths working so `legacy/index.html` remains locally viewable
- Treat archived markdowns as source material, not disposable boilerplate
